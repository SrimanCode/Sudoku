Sudoku Solver and Validator

Welcome to the Sudoku Solver and Validator! This is an interactive application designed for Sudoku enthusiasts. The primary goal of this project is to provide an easy and convenient platform for users to validate and solve Sudoku puzzles. This application is designed to be compatible with Sudoku boards from LeetCode. You can easily copy Sudoku boards from LeetCode, paste them here, and check if they can be solved. Give it a try, enjoy solving Sudoku puzzles.
